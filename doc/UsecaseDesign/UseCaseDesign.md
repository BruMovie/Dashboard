# 一、用例设计

- Boundary类

  - SearchPage:搜索界面，可以搜索电影和影院
  
  - OrderPage:订单界面，可以进行座位的选择
  
  - PayPage:支付界面

- Controller类

  - systemcontrol:控制系统

- Entity类

  - Movie
  
  - Cinema
  
  - Seat
  
  - Order
