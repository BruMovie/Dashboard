# 系统分析与设计-竞对调查-淘票票
1. 公司：淘宝软件有限公司
2. 后台：阿里巴巴
3. 软件定位：泛娱乐产业营销平台，业务范畴由电影扩展至演出、体育等泛娱乐业，自身定位由线上售票平台转变为具备线上线下双向宣发能力的营销平台。
4. 主要功能：
	- 买票
	- 预告片
	- 高清海报
	- 剧照
	- 剧情介绍
	- 网友评论
